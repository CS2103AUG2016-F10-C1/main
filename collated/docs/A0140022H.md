``` md
#### Listing tasks : `ls`
Lists all tasks. 
Format: 
* `ls`
* `ls /dt [dsc]`
* `ls /p [dsc]`

> All tasks listed by default.
> 
> Use /dt to list all tasks by earliest end dateTime.
> 
> Use /p to list all task by priority from low to high.
> 
> Use dsc with previous two prefixes to reverse the order.

Examples:
* `ls`
* `ls /dt`
* `ls /dt dsc`
* `ls /p`
* `ls /p dsc`

```
